Mit externem RIS

Via OParl-Schnittstelle

Ihre Kommune nutzt ALLRIS, regisafe, Somacos oder ein anderes RIS mit OParl? Kein Problem:

  • Automatischer Import öffentlicher Daten
  • Alle Work-Features nutzbar
  • Unabhängig vom RIS-Anbieter
  • Kein Wechsel des RIS nötig

Hinweis: Nicht-öffentliche Vorlagen können manuell hochgeladen werden

Contributing to OParl

Wir setzen uns aktiv dafür ein, unsere Erweiterungen in den OParl-Standard einzubringen. So profitiert die gesamte Community von neuen Features - nicht nur Mandari-Nutzer*innen.

Keine Anbieterabhängigkeit

Ihre Daten gehören Ihnen

Anders als proprietäre RIS-Lösungen setzen wir auf ein offenes Ökosystem. Alle Daten können jederzeit exportiert werden - maschinenlesbar und vollständig.

  • Vollständiger Datenexport

    JSON, CSV, OParl - Sie wählen das Format.

  • Offene Datenschnittstelle

    Dokumentierter Zugang für eigene Anwendungen.

  • OParl-Standard

    Import und Export im offenen Standard.

  • 100% Open Source

    AGPL-3.0 Lizenz - volle Kontrolle.

API-Beispiel 
# Alle Sitzungen exportieren
curl https://api.mandari.de/v1/meetings \
  -H "Authorization: Bearer $TOKEN"

# OParl-Export
curl https://api.mandari.de/oparl/body/1

# Vollständiger Daten-Export
curl https://api.mandari.de/v1/export \
  -o kommune-backup.json
